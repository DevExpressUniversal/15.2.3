To display the Web Document Viewer, call its OpenReport or OpenReportXmlLayout method.

To be able to run the Web Document Viewer, the application must reference all necessary JavaScript libraries (i.e., jQuery, jQuery-UI, KnockoutJS, and Globalize),
or have the EmbedRequiredClientLibraries option set to true (either in the application's Web.config file, or by using the corresponding static property).
